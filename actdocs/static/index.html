«Saint Perl 2014» — шестая в Санкт-Петербурге тематическая конференция, посвященная языку программирования Perl. Мероприятие пройдёт 20-21 декабря 2014 года.

20 декабря впервые в рамках Saint Perl будет проведён хакатон, тема которого — «Высокоуровневый интерфейс к Git из Perl». Git — отличная и многими любимая система контроля версий, но из-за его внутреннего устройства программная интеграция с ним нетривиальна. Решить эту проблему призвана библиотека libgit2.

Мы посмотрим, какие существуют решения для работы с Git из программ на Perl, чем они хороши и чем плохи, и попробуем найти решение для имеющихся проблем.

Хакатон пройдёт по адресу: ул. Воронежская, д. 5, офис "EPAM Systems". Начало хакатона 11:00, телефон для связи +7(911)7767108.

21 декабря — традиционный день докладов. Если вам есть, что рассказать — ждём ваших заявок.
